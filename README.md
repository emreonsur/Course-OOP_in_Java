## Purpose of This Repository

This repository serves as a comprehensive record of my study and practice in the **"OOP with Java"** course. I use this repository to:

1. **Translate and Copyedit Notes**
    - Original notes are gradually translated from Turkish to English to make the content accessible to an international audience while enhancing my understanding of the concepts.
    - Copyediting the notes ensures clarity and accuracy.
2. **Solve Homework**
    - All assignments and homework are solved, and solutions are systematically added here for reference.
3. **Active Practice through IntelliJ IDEA Project**
    - I have created an additional IntelliJ IDEA project in the `Emre_Onsur-Practice_Hub` folder which can be found under the `src` folder where I rewrite the course's original code and classes, and projects. This hands-on approach allows me to assimilate the content and practice coding effectively.

## About the Course

The "OOP with Java" course focuses on developing strong foundations in Object-Oriented Programming principles using Java. It includes topics such as:

- Core OOP principles (Encapsulation, Inheritance, Polymorphism, and Abstraction)
- Java-specific syntax and features
- Real-world application design with Java
